## 목적

PRD에서 명시한 5대 사이트 중 **국민건강보험**(경력증명서 검증용 건강보험자격득실확인서 진위확인)이 랜딩페이지 "실제 기관 사이트에 이렇게 자동 입력됩니다" 섹션에 누락되어 있었음. 이를 추가하여 실제 제품 스코프와 일치시킨다.

## 구현 내용

### HTML
- `#sec-sites > .sites-tabs`에 6번째 탭 추가: `🏥 건강보험 / 자격득실확인서`
- `onclick="switchSiteTab('nhis',this)"`

### JavaScript
- `SITE_DATA` 객체에 `nhis` 키 추가
  - `title`: 🏥 국민건강보험 — 자격득실확인서 진위확인
  - `url`: si4n.nhis.or.kr / 증명서 진위확인
  - `desc`: 국민건강보험공단 증명서 진위확인 서비스. 발급번호·생년월일·성명으로 경력증명서 검증
  - `fields`: 문서종류, 발급번호, 생년월일, 성명
  - `note`: 실제 연동: si4n.nhis.or.kr

### 이미지
- `src/assets/건강보험자격득실확인서 진위확인 페이지.png` → base64 인코딩하여 `IMG_NHIS` 상수로 인라인 삽입
