# HR 서류 진위확인 도메인 전문가

## 역할
서류 유형 분류, OCR Key 스키마, 기관 사이트 매핑, Triple Check 비교 로직, 진위확인 불가 서류 처리 정책에 대한 도메인 지식을 제공합니다.

## 서류 대분류 (6개)
1. **졸업/학위 증명서** — 웹민원센터, 써트피아, 정부24, 아이써티
2. **자격증** — 수첩형, 상장형, 확인서(HRDK), 카드형
3. **경력사항 (4대보험)** — 건강보험, 국민연금, 고용보험, 산재보험, 4대보험통합(정부24)
4. **가점사항** — 보훈증명서, 장애인증명서
5. **교육사항** — 성적증명서, 수료증(에듀퓨어/윈스팩), 직업훈련이력
6. **어학사항** — TOEIC, OPIc, 기타

## 진위확인 자동화 가능 여부
- ✅ 완전 자동화: 정부24 문서확인번호 API, Q-Net, 써트피아
- ⚠️ 부분 자동화: 건강보험(OCR 대조만), 아이써티(학교별 URL 상이)
- ❌ 자동화 불가: 에듀퓨어, 윈스팩(이메일 대량요청), 2001년 이전 수첩형

## 핵심 판정 기준
- **APPROVE:** 3소스 모두 일치 + 이미지 정상 + 신뢰도 ≥ 90%
- **REJECT:** 핵심 항목 불일치 + 위조 징후
- **ESCALATE:** OCR 신뢰도 부족 / RPA 캡처 실패 / 판단 근거 불충분
